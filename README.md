# Wyckoff Trading Agent + A档异动扫描

本项目包含**两个完全独立的交易策略**：

## 策略架构（2026-06-16 重构）

### 1. 威科夫策略（Wyckoff）
- **定位**：中长线结构驱动
- **目录**：`strategies/wyckoff/`
- **核心逻辑**：阶段识别 + 经典结构点（Spring、SOS、UTAD、SOW 等）
- **适合场景**：趋势行情、较长周期回测

### 2. A档异动扫描策略（Anomaly）
- **定位**：短线异动追单
- **目录**：`strategies/anomaly/`
- **核心逻辑**：多因子异动打分（OI异动、主动买卖盘、价格异动、成交额异动等）+ 结构标签生成
- **适合场景**：短线异动、爆发行情

两个策略**数据源相同**（Bitget 公开行情），但**信号生成、参数、回测完全独立**，便于后续多策略对比与融合。

## 如何单独回测 A档异动策略

```bash
npx tsx scripts/run_anomaly_backtest.ts BTCUSDT 1day 300
```

支持的参数：
- 交易对（BTCUSDT / ETHUSDT / SOLUSDT 等）
- 周期（1day / 4h / 1h / 15min 等）
- K线数量（建议 200~500）

示例输出：
```
=== A档异动扫描回测 ===
交易对: BTCUSDT
周期: 1day
K线数量: 300

共生成信号: 2 个

1. 2026-05-12T00:00:00.000Z | LONG | 分数: 68 | 标签: 主动买领先
2. 2026-06-03T00:00:00.000Z | LONG | 分数: 71 | 标签: 持仓异动
```
